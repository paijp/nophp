  • This is a sample to be selected from the list and modified.
  • The "`:bq`id__:g`:bq`" is the string "id" and the command ":g".
  • The command ":g" takes a GET parameter (the ?id= part of the URL) of the specified string and turns it into a number.
  • Therefore, if "?id=" is not specified, it will be an empty string, and "<!--{if-->", "<!--}{else-->" and "<!--}-->" will be executed and the second half of the string will be listed or displayed.
  • If "?id=" is specified, the first half is enabled and you will be taken to a screen where you can modify the contents.
  • In this case, "<!--{tableid message0 `:bq`id__:g`:bq`-->" to specify the record ID.
  • The TEXTAREA tag looks at name and automatically expands the body field that was in the record with the specified ID.
