  • Unlike the last time, the command to set the login ID in the owner field is written in the hidden name, not in the submit.
  • If you wrote a field name in name=, it was saved in that field, but if you write a command in name= and output it to the database, it will be saved on submit.
  • If you write a command in name=, the field value will not be output to value= (even if you edit an existing record by specifying the ID in tableid).
  • Although not used in this case, you can specify a command (e.g., value="`:bq`owner__:r`:bq`") for value= to display the contents of the record.
