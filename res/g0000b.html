`created__:r__y/m/d H:i:s__:todate` 
`body__:r`
  • The record is created with the current time in created, and modified with the current time in created and modified, respectively.
  • The current time is the number of seconds (i.e., the number of digits) elapsed since January 1, 1970 09:00 JST.
  • With `:bq`created__:r`:bq`, this number of seconds is obtained.
  • After that, "__y/m/d H:i:s__:todate" converts seconds into numbers such as years and hours.
